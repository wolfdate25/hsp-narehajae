## :key: How to Use

### Prerequisites

- [Node.js](https://nodejs.org/ko/download/)

### Installation

1. Clone this repo

```
git clone https://github.com/yuseon-Lim/hsp-raonhajae
```

1. Install all npm packages

```
npm install
```

3. Run website. (By default, it runs on **localhost:3000**)

```
npm start
```
